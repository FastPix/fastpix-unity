# DirectUploadVideoMediaMp4Support

Generates MP4 video up to 4K ("capped_4k"), m4a audio only ("audioOnly"), or both for offline viewing.



## Values

| Name                | Value               |
| ------------------- | ------------------- |
| `Capped4k`          | capped_4k           |
| `AudioOnly`         | audioOnly           |
| `AudioOnlyCapped4k` | audioOnly,capped_4k |
